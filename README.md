📌 Project Overview

I built a Slot Machine Game using Python as a mini project to strengthen my understanding of Python programming fundamentals. The project simulates a real slot machine where users can deposit money, place bets, spin the machine, and win rewards based on matching symbols.

This project helped me practice problem-solving, modular programming, and working with Python data structures.

🚀 Features
💰 Deposit money before playing
🎯 Choose the number of betting lines
💵 Place bets on each line
🎲 Random slot generation for every spin
🏆 Winning calculation based on matching symbols
📊 Balance updates after every game
🔄 Play multiple rounds until the user quits
✅ Input validation to prevent invalid entries
